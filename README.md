# 台灣天氣預報網站範例

這個專案是一個簡單的前端範例網站，用來展示：

- 台灣城市/縣市天氣因子查詢
- 中央氣象局開放資料 CWA API
- 預報因子下拉選單
- 每日外出適宜分數判斷

## 範例檔案

- `index.html`：網站主頁面結構。
- `styles.css`：頁面樣式與排列設計。
- `script.js`：CWA API 呼叫、因子選單、預報顯示與外出適宜分數計算。

## 使用方式

1. 把 `CWA_API_TOKEN` 填入 `script.js`。你可以申請中央氣象局開放資料授權碼。
2. 在本機開啟 `index.html`。
3. 從下拉選單選擇台灣城市/縣市與預報因子。
4. 系統會呼叫 CWA API `F-D0047-091`，顯示該因子的預報資料與每日外出適宜分數。

## 功能說明

- `index.html`：定義整體頁面區塊與因子選單。
- `styles.css`：負責背景色、卡片樣式、表單與響應式排版。
- `script.js`：
  - `requestWeather(city, factor)`：呼叫 CWA API `F-D0047-091` 並解析預報資料。
  - `renderForecast()`：顯示所選因子的時間區間預報值。
  - `renderScore()`：根據因子結果計算每日外出適宜分數。

## 拓展建議

- 加入更多因子選項，例如相對濕度、紫外線指數或裸露風險。
- 改為多因子評估，讓外出分數綜合溫度、降雨與風速。
- 加入圖表視覺化每日分數趨勢。
- 將此範例改成 React / Vue / Svelte 並串接後端以統一資料格式。
